How It Works

Three steps from install to first estimate.

1

Run Setup

Execute setup-aspen-brain.bat once. It installs files, creates a START-ASPEN-BRAIN desktop shortcut, and launches the server.

2

Open Admin Panel

Click the desktop icon or navigate to /admin.html and sign in with password aspen2026 to configure pricing.

3

Build Estimates

Select services, choose Economy / Standard / Premium, and share the client-facing total instantly.

Pricing Formula
Sell Price = (Material Cost × Strategy Markup) + (Labor Hours × Shop Rate) + Trip Fee
Economy — 30%
EV Charger: ($450 × 1.30) + (3h × $120) + $75 = $1,020
Standard — 45%
EV Charger: ($450 × 1.45) + (3h × $120) + $75 = $1,088
Premium — 60%
